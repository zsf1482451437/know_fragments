# 动画相关

## hover放大

```css
.list-img {
	transition: all 0.3s;
}
.list-img:hover {
	transform: scale(1.05);
}
```



## hover到元素，兄弟有动画

.el-link和i是兄弟关系哦

```css
// 父元素hover，子元素向右移动动画
.hover-move {
	&:hover {
		.icon {
			// hover到父亲，icon选中的元素执行过渡动画
			animation: zsf 1.5s ease;
		}
		@keyframes zsf {
			0% {
				transform: translate(5px, 0);
			}
			100% {
				transform: translate(20px, 0);
			}
		}
	}
}

```
