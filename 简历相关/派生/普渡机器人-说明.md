# 平台工具前端 JD 匹配说明

## JD 关键词

- 平台型、工具型产品研发经验
- Vue/React、HTML5、CSS3、JavaScript/TypeScript
- 前端工程化、模块化、组件化设计
- 前端性能优化
- 大数据量渲染：虚拟列表、Canvas/WebGL
- 可视化方向：Three.js/WebGL、点云、视频帧级标注、ECharts/D3
- 独立 Owner 模块从 0 到 1 建设

## 匹配策略

- 主轴：平台/工具型前端 + React 工程化 + 复杂列表/性能优化。
- 强化：`Shopify 自研应用`、`AI Agent 电商系统`、`Reolink 智能安防App`、`GogoalShop 跨境商城`。
- 弱化：纯活动页搭建、纯展示型商城页面的篇幅，避免简历看起来只是页面开发。
- 风险：JD 中 5 年经验、Three.js/WebGL、点云、视频标注等方向当前缺少直接项目证据，不写成直接落地经验。
