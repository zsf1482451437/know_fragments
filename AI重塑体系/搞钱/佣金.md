# 京粉

1. 下载京粉 APP，用京东账号一键登录。  
2. 在京粉搜索“春节送礼”“年货礼盒”等关键词，筛选京东配送、高佣金、近 7 天热销、评分 4.9+ 的商品。  
3. 收藏合适商品，记录原价、券后价、佣金和发货时效，整理成选品表，方便后续上架。  
4. 上架闲鱼时，直接参考京粉现成模板，或借鉴同行的标题、主图和文案，快速发布。  
5. 有客户咨询或下单时，把京粉推广链接发给对方，客户通过链接成交后即可获得佣金。  
6. 为提升转化，可以把部分佣金做成礼金优惠券发给用户，在京粉后台「营销工具」中即可设置。

# 短剧推广

1. 先确定推广平台，比如抖音、快手、小红书、视频号等，优先选当前流量高、分佣明确的平台。  
2. 选短剧时重点看题材和转化，比如霸总、逆袭、家庭、悬疑这类高点击题材，优先挑完播率高、素材多、佣金清晰的剧。  
3. 拿到授权素材后，整理成选剧表，记录剧名、题材、爆点、分佣比例、结算方式和可投放平台。  
4. 发布内容时，直接围绕高冲突片段做切条，标题突出反转、情绪和悬念，文案重点引导用户继续看或点击入口。  
5. 用户通过你的挂载链接、口令或小程序入口进入并完成充值、解锁或观看后，你就能拿到对应佣金。  
6. 为提升转化，可以多测不同封面、开头 3 秒和标题文案，优先放大高点击、高转化的素材，持续迭代爆款内容。  

# 京粉/淘客/返佣流程

1. 先确定平台和账号体系，比如京粉、淘宝联盟、拼多多返利、短剧 CPS 等，优先选择结算规则清晰、素材现成的平台。  
2. 进入平台后先选品或选项目，重点看佣金比例、转化率、发货时效、售后难度和平台限制。  
3. 把有潜力的商品或项目整理成表格，至少记录链接、原价、券后价、佣金、卖点、适合人群和推广渠道。  
4. 根据不同渠道生成推广链接、口令、二维码或小程序卡片，确保每个渠道都能单独追踪效果。  
5. 在闲鱼、社群、小红书、抖音、朋友圈等场景发布内容，持续引导用户点击链接或咨询下单。  
6. 用户成交后复盘数据，看点击、咨询、下单和退款情况，持续优化选品、文案和优惠策略。  

# 佣金玩法对比

- 京粉：适合实物电商，优点是京东配送和信任度高，缺点是佣金波动和选品竞争较强。  
- 淘客：商品池大、佣金玩法成熟，适合大规模选品，但平台规则更复杂，容易卷低价。  
- 返佣：适合做省钱心智和长期用户沉淀，复购空间大，但需要更强的私域承接能力。  
- 短剧推广：起量快、爆发力强，适合内容型流量玩法，但素材迭代快、平台审核更敏感。  
- 总结：实物类更适合稳定出单，内容类更适合快速放大，私域返佣更适合长期复购。  

# 选品 SOP

1. 先选赛道，比如礼盒、家清、零食、美妆、数码配件、短剧题材等，优先选高需求和高复购方向。  
2. 再看数据，重点关注近 7 天销量、佣金比例、券后价、评分、退款率和同类竞争强度。  
3. 再看内容，确认有没有现成主图、视频素材、标题模板、达人案例和真实买家反馈。  
4. 再看交付，确认发货时效、售后难度、平台限制和是否容易被投诉。  
5. 最后做表格分级，把商品分成主推款、测试款、备用款，避免一上来铺太多。  

# 转化话术

- 低价刺激：这款现在有券，券后价更低，今天下单最划算。  
- 稀缺提醒：这个活动价和佣金窗口都比较短，拖一拖可能就恢复原价。  
- 场景带入：如果你是送礼、自用、囤货，这类款最容易出单。  
- 降低决策：我已经帮你把链接和优惠都整理好了，直接点进去下单就行。  
- 提升信任：这个是平台官方链接，不是私下转账，正常下单就能享受优惠。  
- 促进成交：如果你现在下，我可以再帮你看看有没有额外券或礼金能叠。  

# 平台规则和注意事项

- 不要虚假宣传，尤其不要夸大佣金、收益、效果或售后承诺。  
- 不要违规搬运素材，特别是短剧、品牌图、达人视频，尽量确认授权范围。  
- 不要私下收款代拍，优先走平台链接，减少纠纷和跑单风险。  
- 不要只看佣金高低，还要看退款率、售后成本和账号安全。  
- 不同平台对外链、二维码、导流、敏感词限制不同，发内容前先确认规则。  
- 所有玩法都建议做数据记录，持续复盘点击率、咨询率、成交率和退款率。
